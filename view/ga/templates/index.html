Welcome to GA's instance of VocPrez, a read-only web delivery system for Simple Knowledge Organization System (SKOS)-formulated RDF vocabularies.

VocPrez presents different views of vocabularies from a list of RDF sources. This system currently implements sources from local files, Research Vocabularies Australia and VocBench.

See the About page for more information about the VocPrez system and its use.

Starting Points

Geoscience Australia conceives of vocabularies as being synonymous with SKOS' Concept Scheme class, so each GA vocab is an instance of skos:ConceptScheme. Not all users of SKOS make this conception.
