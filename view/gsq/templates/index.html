Welcome to an instance of VocPrez, a read-only web delivery system for Simple Knowledge Organization System (SKOS)-formulated RDF vocabularies.

VocPrez presents different views, or profiles, of vocabularies which it can get from a number of different RDF sources, such as local RDF files, Research Vocabularies Australia (a vocabulary management system), the VocBench too or a SPARQL endpoint.

See the About page for more information.
